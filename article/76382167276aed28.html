五·七·五的少女世界

漫画《川柳少女》推荐

Comic|chitoge2018年10月30日 6时15分

  • 书名 ：《川柳少女》/ 「川柳少女」
  • 作者 ：五十岚正邦
  • 日本出版 ：连载于《周刊少年Magazine》 ，讲谈社出版6卷
  • 中文出版 ：暂无

在过去，Anitama 推荐过 《默绘女高》 这部漫画。这部漫画以篇幅较短、没有台词的高中女生的日常生活为卖点，故事简单诙谐，为读者提供驻足歇息之地。有无声默剧的艺术，也有讲究说话的艺术、以日本传统一隅为主题的《川柳少女》。

漫画单行本第2卷封面。

《川柳少女》的主角名叫雪白七七子，因觉得难以言简意赅地说出自己想说的话，便使用书写的形式传达自己的所思所想。比较特殊的是，七七子在纸板上写下的，是依照日本传统诗的川柳句式规律的留言。

川柳与俳句一样，同为17个音节，具体以五·七·五的顺序排列。因其韵律和谐、内容精简，深受日本人的喜爱。七七子这位女高中生使用川柳交流，古典传统与可爱的JK形象相结合，令人眼前一亮。

俳句要求必须出现一个代表季节的词语，通常创作的内容饱含创作者的情思。而川柳则没有此限制，并且门槛较低、贴近生活，容易让人感同身受、有所兴趣。《你的名字。》在过去两年前大受好评，“君の名は。”也成为了当时流行语之一，在 2017年的上班族川柳比赛中也有以“君の名は。”为主题的作品，并且在最终投票评选中名列前茅：

久しぶり！ / 聞くに聞けない / 君の名は
那么久不见！/ 心里想问却没问 / 你的名字是

ありのまま / スッピンみせたら / 君の名は？
展现真实的 / 没有化妆的素颜 / 你的名字是？

君の名は / ゆとり世代の / 名が読めず
你的名字是 / 宽松世代的名字 / 我不会读啊

《川柳少女》漫画官方在活动方面也不甘示弱。在连载开始就推出了与读者互动的环节，读者可在调查回执或寄件中，附上自己创作的川柳，编辑部会挑选优秀的川柳作品刊载在杂志上。这个活动虽然看似简单，但增强了与读者的互动性，不仅将这个传统文化延续下去，而且也使得读者可以从创作的角度，感受作中出现的川柳语句，可谓是一石三鸟。

此前《周刊少年Magazine》应募川柳投稿的推文。

聊完“川柳”聊“少女”。七七子性格开朗，足够天然，长相甜美可爱，黑长直搭配黑裤袜，可谓是经典搭配，一直都受到广大读者观众的喜爱。该作以七七子的高中日常生活为主题，讲述七七子身边的人和事。

在川柳少女的日常中，原作者五十岚老师也不忘点缀上些许粉红色。与七七子一起研讨川柳的伙伴名叫毒岛英二。英二因为觉得随口念出来的川柳句极其顺口，便一心投入川柳的学习中，不再作为不良少年度过充满暴力的日子。

两人的相遇是在圣诞节的聚会上，因为英二的川柳不受拘束、诙谐幽默，七七子在倍感欢喜的同时也逐渐在意起英二的每一件事。两人最早的放闪便是在连载第2话时，英二因为在意自己凶恶的形象会让人误会七七子，结果七七子生气着写道（没有文采的我，选择了直译）：

構（かま）わない / どう思（おも）われても / きみとなら
没有关系的 / 不管别人怎么想 / 若有你相伴

摘自漫画第2句。

并没有发觉这句话的特别性的七七子，在英二指出之后羞涩地连忙挥手否认。五十岚老师以平均每2至3回画上一篇两人间的甜蜜互动，有时候也会来一次蜜糖连发，其甜度多次引起读者们的“不适”。

好吃的甜点，也需要其他滋味调剂甜度。在两人身边的，有搞笑满点、经常秘密监视七七子与英二两人的文艺部部长片桐天音，她的另一身份是以 “北金麻理亚”为笔名的小说家。有英二幼时的玩伴，身材在众角色之中鹤立鸡群的大月琴学姐，以调戏英二七七子等人为乐，是 “美国陆军格斗术”部的部员。还有与七七子相似、使用图画展现自己表情的美术部学生矢工部纪，因为自己是独生女，所以憧憬着七七子这样一般的姐姐的存在。

这几位角色在五十岚老师的笔下，圆圆的脸型更显画风的清新可爱，搭配着欢快轻松的故事，可谓又是一篇能够让读者身心放松的读物。

但是，如果故事千篇一律，没有别的发展，即使是让读者“放松”的作品，也很容易让读者产生“疲劳感”。于是随着漫画逐渐进展，五十岚老师创作的全新角色接连登场，使得故事不再一如既往地只是充满欢笑的校园日常。

摘自漫画第102句。

比如在第102句与七七子的正面接触的，七七子过去的同学。她直言七七子现在的状况反而会让自己喜欢的人被人看不起，从此之中，也可以窥得之后的发展。虽然本作以川柳为主题元素，但仅凭着五七五的书面表达以及天然的性格，七七子的存在是格格不入的。

摘自漫画第60句。

与名为现实的怪物的战斗不能逃避，感情上也遭遇了敌手。过去与英二是同学，虽然是文学少女、但因为对英二有好感，为了吸引英二注意、打扮成不良少女的明司五町在单行本第4卷（第59句）的末尾登场，她的主动进攻会让她、英二与七七子之间的故事有更多的未知性。

在此之中，七七子要怎么样迈出新的一步获得成长，其余角色又会在故事上带来怎么样的可能性，便是之后值得期待的内容。

《川柳少女》连载已有2年，作品内容比起同刊连载的其他作品来说比较平淡如水，能够以四格的形式在周刊少年杂志中站稳脚跟，拥有一批忠实读者，实属不易。或许《川柳少女》能够在有限的篇幅最大地展现自己的魅力，在内容繁杂的杂志中给予读者休憩之地，便是它受欢迎的原因吧。

原作者五十岚正邦为七七子参选《周刊少年Magazine》女主角人气投票绘制的角色视觉图。

有时候，我们也会一时兴起，对自己写下的句子斟酌推敲，只为让句子听起来更加中意。可能看起来是一件极具学术性的事情，但《川柳少女》却以别样的风格，展现了川柳与生活相融的趣味。不如学着书中的七七子与英二，试着创作几句诗词，也许你也能体会到未曾感受过的乐趣。

封面： 《川柳少女》
