当amazarashi遇到Aimer

两组超强音乐人的首度联手亚洲巡回，首战上海！

PR|AMUSE中国2017年12月14日 5时00分

被中国粉丝誉为“音乐诗人”之名的amazarashi，携手日本实力歌姬Aimer即将开启“amazarashi × Aimer Asia Tour 2018”亚洲巡回首站，于2018年3月3日、4日在万代南梦宫上海文化中心（梦想剧场）举办对盘演唱会！

相信对于喜欢音乐的中国粉丝来说，这两组艺人的名字应该不会陌生，amazarashi这支特立独行的乐队的歌曲一直以来都被中国粉丝们熟知，包括他们为中岛美嘉所做的《僕が死のうと思ったのは》、以及《季节は次々死んでいく》（《东京食尸鬼第二季》片尾曲）、《空に歌えば》（《我的英雄学院》OP曲）等多首耳熟能详的名曲，而真正让他们被粉丝所喜爱的则是他们音乐中写实又奇特的视角，仿佛他们可以用音乐抓住人心中最脆弱的部分；曾在上海等亚洲各地举办过巡回演出的他们，这次也会通过自己的音乐向全亚洲他们的粉丝传播独特奇妙的世界观；

Aimer则在今年7月作为BILIBILI MACRO LINK - STAR PHASE x ANISONG WORLD MATSURI 2017的出演嘉宾 登上上海梅赛德斯-奔驰文化中心的舞台惊艳亮相。对音乐从小有极高领悟力的Aimer，她干净舒服的声线总能不由让人驻足聆听。一出道就备受瞩目的她不仅和泽野弘之等音乐大师合作，各路知名音乐人如TAKA（ONE OK ROCK）和野田洋次郎（RADWIMPS）也为她度身制作歌曲。时而如《LAST STARDUST》（剧场版Fate/stay night [UBW]插曲）二次元燃向、时而如《蝶々結び》一般的日系清新、时而如《茜さす》（《夏目友人帐 伍》主题歌）催泪温暖，时而如《花の唄》（剧场版「Fate/stay night [Heaven’s Feel]主题歌）一般展现出宽广的音域.多变的歌曲诠释也使得Aimer成为近年来日本歌坛备受瞩目的日本女歌手。

相信对于这两组艺人的粉丝来说，会希望有更加机会可以领略到他们的舞台上的精彩吧。这两组你绝对意想不到的强势梦幻组合，将在上海万代南梦宫梦想剧场联手举办2天live,，一起开启amazarashi 和Aimer的音乐之旅吧。

演出详情如下：

2018年3月3日(周六) OPEN 17:30 / START 18:30
2018年3月4日(周日） OPEN 16:30 / START 17:30

场地：万代南梦宫上海文化中心 (梦想剧场)
地址:上海市普陀区宜昌路179号

票价、特典等其他详情将于近日发布，尽情期待

来自amazarashi和Aimer的留言

【amazarashi秋田ひろむ】

「Aimerさんとのアジアツアー決定しました。新しい出会いも再会も、新しい場所も再訪も、成功も失敗も、楽しいも悔しいも、どんな新しい感情に出会えるか今から楽しみにしています。自分の国の外に、僕らの歌がどう響くのか、もしくは響かないのか、肌で感じられたら嬉しいです。 」

“我决定和Aimer一起举办亚洲巡回，所以非常期待接下来会不会有新的邂逅或再遇，初涉新地或再访故地，成功或失败、开心或后悔，或者其他的新的感情。并且在自己国家以外的地方，很非常开心可以切身感受到自己的歌曲会不会引起反响。”

【Aimer】

「わたしにとって初めてのアジアツアーを、amazarashiさんと旅できることが光栄です。海を隔てた遠い地で秋田さんの歌、その詞にどう心抉られるのか、その中で自分がどんな歌を歌えるか、とても楽しみです。」

“对于我来说的首次亚洲巡回可以和amazarashi一起开始旅程深感荣幸。在隔海遥远的土地上，秋田先生的歌和歌词如何打动人心，而自己本身会唱怎样的歌曲，都非常期待”

封面： Aimer
