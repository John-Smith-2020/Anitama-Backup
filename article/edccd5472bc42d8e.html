《新干线变形机器人 辛卡里昂》与《新世纪福音战士》联动的新干线将于剧中登场

目前正在热播中的《新干线变形机器人 辛卡里昂》与《新世纪福音战士》联动的新干线“500 TYPE EVA”将于剧中登场。

“500 TYPE EVA”是由庵野秀明监修、机械设计山下いく设计的电车，于2015年于JR西日本山阳新干线联动的样式，今年5月13日为止一天一趟往返于新大阪—博多1日。也实现了辛卡里昂与500 TYPE EVA在玩具上的联动，“500 TYPE EVA”在动画中以怎样的形式登场目前暂未公开，详细等续报发表。
