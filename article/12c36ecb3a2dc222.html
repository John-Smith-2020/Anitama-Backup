Fun Q

Anitama新声

Voice|谢枫华2017年7月12日 21时00分

Shaft 制作的岩井俊二原作改编动画电影《烟花》原本预定在昨天举办完成报告见面会，然而东宝的制片人在活动上表示：“制作这部电影的是 Shaft 这家优秀的工作室，但是他们会跳票这一点也很有名，如今也正在坚持不懈地进行制作，结果就是，原本应该在今天举办的完成报告见面会变成制作报告见面会了。”

（https://twitter.com/botacou/status/884611902789124096）

演出家西村大树突发奇想：如果让我的丁丁在有很多美少年的动画里登场，是不是也会被女孩子们当做圣地大为兴奋呢？

（https://twitter.com/taiki_nishimura/status/884677868566962176）

考虑到面向男女性的动画数量比例，我觉得让屁股在有很多美少女的动画里登场可能会更快。

衍生自吉冈平《无责任》系列小说的原创短篇动画《无责任银河☆泰勒》在昨晚开始播出。这一系列小说在 90 年代推出过以《无责任舰长》为题的动画，动画由龙之子制作，对小说进行了极大幅度的改编，修改、原创了许多设定，最终大获成功。

但是，《无责任舰长》的机体设计之一伊藤浩二看到新动画《无责任银河☆泰勒》，却吃了一惊：新动画里，出现了他为动画设计的机体。由于新动画制作公司并不是《无责任舰长》的龙之子，而是 Seven，伊藤对此便感到非常困惑，不知道这种做法是否有版权问题。他说，如果要在动画里以那种恶搞的方式使用他的机体，至少希望能打个马赛克。

伊藤对同行西村诚芳打了个比方：如果别的《高达》系列作品里，使用了西村设计的角色，他感觉如何？西村也赞成说，哪怕自己的角色只出现了一卡，也希望对方能事先和自己打个招呼，更何况这次《无责任》的动画还换了制作公司。

动画人高木弘树则想起，他参与的《DOMINION》系列也发生过类似的问题。J.C.STAFF 制作的外传动画《特搜战车队DOMINION》，照抄了 Agent 21 制作的前传《DOMINION》的设定。

（https://twitter.com/manakare_i/status/884768043254992900）

看样子，就算是衍生自同样的原作，由不同制作公司、不同创作者设计出的原创形象的权利归属，也仍然是一个争论不清的问题。

由 Studio Khara、麻生墅和 Dwango 三家公司在九州岛福冈市共同设立的新动画制作公司 Project Studio Q，在今日举办记者见面会。新公司由 Dwango 董事太田丰纪兼任社长，庵野秀明出任创作管理统括。公司名字中的“Q”来自于九州（Kyuushuu）的谐音和 Quick（高速）、Quality（品质）的首字母。

庵野秀明表示，在 20 年前，动画出于冲洗和制作进行的需要，还只能在东京制作。然而如今，动画制作已经没有必要拘泥于东京，有必要在东京以外创造能够成为据点的地方。而福冈从各种意义上都很美味，特别是庵野喜欢博多猪骨拉面，对新公司的夜宵满怀期待。

但是，刚刚设立的新公司一上来就制作自己的作品，十分困难。目前可以确定的是，Project Studio Q 将会参加《福音战士》新剧场版 CG 部分的制作。至于之后的发展，还在规划阶段，要配合招人的进度。

Dwango 会长川上量生 （就是宫崎骏纪录片里被宫老骂的那位） 表示，只要能招到 20～30 人，就可以把企划交给 Q 来做，所以这 20～30 人就是 Q 现阶段的招人目标。希望能在几年后发表以 Q 为主体制作的作品，至于这个“几年”，也要看招人的进展如何。如果够快，可能就是明年的事了。

目前，Project Studio Q 的官方网站上正在公开募集多个职位的人才。今后还会举办竞赛，吸引人才。感兴趣的读者也不妨关注一下。

除了三家出资公司的代表之外，福冈市长高岛宗一郎也出席了见面会。

（https://twitter.com/mnishi41）

封面： 《无责任银河☆泰勒》
